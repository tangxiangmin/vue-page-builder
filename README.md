

页面编辑器
===


## 步骤

编辑器
```
npm i 

npm run dev
```

后台项目
```
cd server

npm i
 
// 初始化表，需要先配置好数据库连接
npx sequelize db:migrate

npm tun dev
```

## Feature

* [ ] 基于文档流和DOM树的移动端页面编辑器
* [ ] 适配Web，后续支持小程序和Flutter
* [ ] 支持动态数据源
* [ ] 支持交互逻辑，如点击事件等
* [ ] 常规内置组件
* [ ] 用户自定义组件库，可以上传自定义组件，可以从用户端下载
